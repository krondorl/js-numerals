<a id="user-content-javascriptfrontend-programmer---excercise-1" class="anchor" href="#javascriptfrontend-programmer---excercise-1" aria-hidden="true"><span class="octicon octicon-link"></span></a>Javascript/Frontend Programmer - Excercise 1</h1>

<p>Fork this project with the original description and write your own code.</p>

<p>Commit the important milestones and not just the final result!</p>

<h2>
<a id="user-content-excercise-description" class="anchor" href="#excercise-description" aria-hidden="true"><span class="octicon octicon-link"></span></a>Excercise description</h2>

<p>Create a frontend application.</p>

<p>We need a web form, which has a numeric input field and a submit button.</p>

<p>When user gives an arabic number, the system shows the english phrase of the number.</p>

<p>For example:</p>

<pre>
7    == seven
42   == forty-two
2001 == two thousand and one
1999 == nineteen hundred and ninety-nine
</pre>

<p>That's all.</p>

<p>Extra points:</p>

<ul class="task-list">
<li>write your code in TDD / BDD, use any test frameworks</li>
</ul>
